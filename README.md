## Installation

1.  Copy the `Contents/mods/` folders into your Project Zomboid `mods` directory (or Workshop content folder).
2.  Ensure `KoniLib` is enabled whenever using `ChatSystem`, `AreaSystem`, `ReactiveUI`, or other dependent mods.
