# AreaSystem

**AreaSystem** is a powerful admin tool for Project Zomboid that allows you to create, manage, and visualize custom zones directly on the in-game world map.

## Features

- **Map Integration** - Visualize areas on the world map
- **In-Game Editors** - Draw zones directly on the map
- **Real-time Sync** - Changes sync across all clients
- **Persistence** - Data survives server restarts

## Quick Start

1. Press `M` to open the World Map
2. Click **"Manage Areas"** button
3. Create and draw your zones

## Requirements

- **KoniLib** (required dependency)
