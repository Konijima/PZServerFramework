# ChatSystem

**ChatSystem** is a custom chat implementation for Project Zomboid that replaces the vanilla chat with a modern, Socket.io-based system.

## Features

- **Multiple Channels** - Local, Global, Faction, Safehouse, Private, Staff, Admin, Radio
- **Command System** - Extensible commands with access levels
- **Proximity Chat** - Local chat respects distance
- **Live Settings** - Sandbox options update in real-time
